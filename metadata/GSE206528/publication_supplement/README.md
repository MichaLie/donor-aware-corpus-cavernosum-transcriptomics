# GSE206528 Publication Supplement Inputs

Some annotation cross-check scripts can use supplementary tables from Zhao et al., Nat Commun 2022, DOI `10.1038/s41467-022-31950-9`.

The third-party supplementary files are not redistributed in this repository assembly. Download them from the source publication or GEO record if you need to rerun those annotation cross-checks from source.
